# af
A (http) fetch CLI 😀👍

## Getting started
`cargo install af`

## Usage
```
af
A (http) fetch CLI 😀👍

USAGE:
    af [METHOD] <URL>

ARGS:
    <METHOD>    HTTP method. If no HTTP method is provided, GET is used by default [default:
                GET]
    <URL>       URL to fetch

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

```
